Using SWIFT to solve puzzles and create a world.

What is Expl. CS2 - Coding 2?

This course takes you through the SWIFT language via a fun game. In the game, you guide a character through certain maze-like structures with your own code. This code can be as lengthy and complicated as you wish, introducing some very intriguing techniques.

My Courses
Creative Coding

Creative Coding is an exploration into programming on the hull. We learn loads of useful programming techniques, including how to debug and create efficient and effective loops. This course also digs slightly into the basics of computer science and a brief understanding of how our everyday electronics function.

Exploring Computer Science 1

Exploring Computer Science 1 is a class made to introduce how a computer works on a low level and how to make a simple, but elegant website. In this course, you will be using HTML & CSS to build a beautiful backbone that will be made even more functional when one eventually learns Javascript. (note: javascript is not taught in this course)

Exploring Computer Science 2 - Coding 2 iOS

This course takes you through the SWIFT language via an engaging game. In the game, you guide a character through certain maze-like structures with your own code. This code can be as lengthy and complicated as you wish, introducing some very fascinating techniques.

Exploring Computer Science 2: Coding 3 iOS

As a continuation of ECS2 - Coding 2, this will extend your knowledge of the subjects and techniques taught in your previous venture. The levels ingame will be a little trickier to navigate than usual, and the loops may take a bit more thought. At the end of this course, you should be capable of solving complex programming challenges one might face in the real world.
